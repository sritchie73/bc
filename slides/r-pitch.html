So for those of you who have never encountered R before, what is it? R is a statistical programming language which is used to: - process and analyse data - visualise data, and generate publication quality graphics It is also a programming language, allowing you to: - develop analysis pipelines - develop new statistical methods It is also a free open-source project, making it completely free to download and extend.

Why R and not <X>?

Prepackaged with common statistics and methods

If something is missing, there's a package for that!

  1. 5000+ add on packages in CRAN
  2. 800+ on Bioconductor
  3. 100+ on github

Cutting edge

So why use or learn R for data analysis, instead of another language or platform? Not only does R come prepackage with pretty much any statistical model or test you can think of, but its also easy to extend and install add-on packages. As a result its used by researchers and industry all over the world, not only for their analyses, but also for method development. This means that R is cutting edge, new methods are often available in R long before they're implemented in other languages or platforms. In November last year, there were more than 5000 statistical and general purpose packages available in CRAN, along with many domain specific packages like those specific to life sciences research. source
