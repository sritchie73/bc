Hello, and welcome to the R section of the Software Carpentry bootcamp.

R is...

An environment for performing statistical analysis

Software for data visualisation

A programming language

Free!

So for those of you who have never encountered R before, what is it? R is a statistical programming language which is used to: - process and analyse data - visualise data, and generate publication quality graphics It is also a programming language, allowing you to: - develop analysis pipelines - develop new statistical methods It is also a free open-source project, making it completely free to download and extend.

Solution

Intensive hands-on training.

Software Carpentry is the solution. We run intensive two-day classes that give scientists and engineers those fundamental skills as part of the Mozilla Science Lab.

Outcomes

20% improvement in productivity is common.

10X isn't rare.

We can often save people 20% of their time - a day a week - and it's not unusual for attendees to report a tenfold savings.
